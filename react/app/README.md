# Odin Visa React Application

This template provides a minimal setup to create an Odin-React App for your project using Typescript.

Remember to install the packages after creating the new app with:
```bash
npm install
```

## Running

Entering the following command will run your Application in Development Mode:

```bash
npm run dev
```

This will run locally on your machine, and be accessible via the localhost port number displayed (usually 5173).
